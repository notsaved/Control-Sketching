# Control-Sketches
Interactive visualization of 450 outputs of the Samsung Sketch-to-Image app, a Galaxy AI feature introduced with One UI 6.1.1 and discontinued with One UI 7. Sketch-to-Image utilizes user sketches to control a proprietary image diffusion model that transforms drawings into images in various styles. Every row of images is prompted with a single sketch (first column) and contains three outputs generated in each of five different styles (Watercolor, Illustration, Sketch, Pop Art, 3D Cartoon). The filterable grid-based visualization allows to compare outputs across control sketches and diffusion styles, highlighting recurring patterns and emergent features precipitated by the probing exercise.
